## 🚀 Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/MoAhmadian/Machine_Learning.git
   cd Machine_Learning
   ```

2. Navigate to the project of interest
3. Open the notebooks (`.ipynb` files) using Jupyter Notebook or JupyterLab
